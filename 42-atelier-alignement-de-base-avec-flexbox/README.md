# Atelier : alignement de base avec Flexbox

**Rappel théorique** Flexbox (Flexible Box Layout) est un modèle de mise en page CSS conçu pour distribuer l'espace entre les éléments d'un conteneur et les aligner efficacement, même lorsque leurs dimensions sont inconnues ou dynamiques. Le principe fondamental repose sur deux niveaux : le **conteneur flex** (l'élément parent auquel on applique `display: flex`) et les **éléments flex** (ses enfants directs). Le conteneur dispose d'un axe principal (main axis) et d'un axe transversal (cross axis). Par défaut, l'axe principal est horizontal (gauche à droite). Les propriétés d'alignement essentielles sont : `justify-content` (aligne sur l'axe principal), `align-items` (aligne sur l'axe transversal pour une seule ligne), `align-self` (surcharge l'alignement d'un élément individuel), et `align-content` (gère plusieurs lignes). Les valeurs courantes incluent `flex-start`, `flex-end`, `center`, `space-between`, `space-around`, `stretch`. Erreurs fréquentes : oublier que `justify-content` et `align-items` s'inversent lorsqu'on change `flex-direction: column` ; confondre `align-items` (une ligne) et `align-content` (plusieurs lignes) ; appliquer des propriétés flex sur l'enfant au lieu du parent. Mini-exemple : ```css .container { display: flex; justify-content: center; align-items: center; height: 100vh; } ``` Ce code centre parfaitement un ou plusieurs éléments au milieu de l'écran, horizontalement et verticalement — l'un des cas d'usage les plus fréquents en développement web réel. --- **Énoncé de l'atelier** Dans cet atelier, vous allez construire une page HTML/CSS autonome qui met en pratique les propriétés d'alignement Flexbox vues dans les chapitres précédents.
