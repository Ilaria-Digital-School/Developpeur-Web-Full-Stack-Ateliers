# Atelier : première media query pour changer la couleur d’arrière-plan

**Rappel théorique** Une media query est une règle CSS conditionnelle qui permet d'appliquer des styles uniquement lorsque certaines conditions relatives à l'environnement de l'utilisateur sont remplies — principalement la largeur de la fenêtre du navigateur (viewport). La syntaxe de base est la suivante : `@media (condition) { /* règles CSS */ }`. Dans une approche mobile-first, on écrit d'abord les styles pour les petits écrans, puis on utilise `min-width` pour enrichir progressivement l'affichage sur des écrans plus larges. Par exemple : `@media (min-width: 768px) { body { background-color: lightblue; } }` signifie que la couleur d'arrière-plan devient bleu clair dès que la largeur du viewport atteint ou dépasse 768px. Les erreurs fréquentes chez les débutants sont : oublier la balise meta viewport dans le HTML (``), confondre `min-width` et `max-width` (le premier s'applique à partir d'une taille, le second en dessous), et écrire des media queries en dehors de la feuille de style liée ou avec une syntaxe incorrecte (parenthèses manquantes, point-virgule mal placé). Le modèle mental à retenir : une media query, c'est un interrupteur — la condition est soit vraie, soit fausse. Si elle est vraie, les styles s'appliquent ; sinon, ils sont ignorés. --- **Énoncé de l'atelier** Dans cet atelier, vous allez mettre en pratique la syntaxe des media queries vue dans le cours précédent en réalisant une page web minimaliste dont la couleur d'arrière-plan change en fonction de la taille de l'écran. C'est un exercice fondamental : maîtriser ce mécanisme de base est la condition sine...
