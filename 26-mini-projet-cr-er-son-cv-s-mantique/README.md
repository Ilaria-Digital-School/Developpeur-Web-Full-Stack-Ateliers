# Mini-projet : créer son CV sémantique

## Description du projet Vous allez créer votre propre CV en HTML en appliquant rigoureusement toutes les bonnes pratiques de structuration sémantique étudiées dans ce chapitre. Ce projet est à la fois un exercice technique et un livrable professionnel réel : à l'issue du mini-projet, vous disposerez d'un fichier HTML propre, accessible et bien structuré que vous pourrez intégrer à votre portfolio de développeur. ## Contexte professionnel En tant que développeur web, la maîtrise du HTML sémantique est non négociable. Un recruteur ou un client qui inspecte votre code source doit y trouver clarté, cohérence et respect des standards. Ce CV sera la preuve concrète de cette maîtrise. ## Compétences validées - Utilisation correcte des balises sémantiques HTML5 (`header`, `main`, `footer`, `section`, `article`, `nav`, `aside`) - Hiérarchie de titres `h1`–`h6` logique et accessible - Intégration des landmarks ARIA pertinents - Indentation propre et commentaires structurants - Nommage de fichiers et arborescence de projet cohérente - Accessibilité de base (attributs `lang`, `alt`, structure de navigation claire)
