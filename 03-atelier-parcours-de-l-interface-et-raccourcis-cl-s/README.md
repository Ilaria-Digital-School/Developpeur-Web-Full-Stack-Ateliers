# Atelier : parcours de l’interface et raccourcis clés

**Rappel théorique** Visual Studio Code (VS Code) est un éditeur de code source développé par Microsoft, aujourd'hui devenu le standard de l'industrie pour le développement web. Comprendre son interface n'est pas une option : c'est la base de votre efficacité quotidienne en tant que développeur. L'interface se décompose en plusieurs zones distinctes : la barre d'activité (Activity Bar) sur la gauche donne accès à l'explorateur de fichiers, la recherche, le contrôle de version (Git), le débogueur et les extensions. Le panneau latéral (Side Bar) affiche le contenu de l'onglet actif. L'éditeur central est l'espace où vous écrivez votre code, avec la possibilité de diviser la vue en plusieurs colonnes. Le panneau inférieur regroupe le terminal intégré, la console de débogage et les problèmes détectés. Enfin, la barre de statut en bas de l'écran affiche des informations contextuelles (branche Git, encodage, langue du fichier). Une erreur fréquente chez les débutants est de naviguer exclusivement à la souris, ce qui ralentit considérablement le flux de travail. Les raccourcis clavier sont précisément conçus pour éliminer cette friction. Par exemple, ouvrir le terminal intégré avec Ctrl+` (backtick) ou basculer entre fichiers ouverts avec Ctrl+Tab sont des réflexes à acquérir dès le départ. Mental model à retenir : VS Code est un cockpit — plus vite vous maîtrisez ses commandes, plus vite vous codez. --- **Énoncé de l'atelier** Cet atelier a pour but de vous faire parcourir méthodiquement l'interface de VS Code et d'ancrer les raccourcis clés essentiels dans votre mémoire musculaire.
