# Atelier : créer une galerie d’images sémantique

**Rappel théorique** En HTML, la sémantique consiste à choisir les balises qui décrivent le mieux le sens du contenu, et non simplement son apparence. Pour les images, deux éléments clés sont à maîtriser : la balise `` avec son attribut `alt` obligatoire (qui décrit l'image pour les lecteurs d'écran et le référencement), et le couple `` / `` qui permet d'associer une image à sa légende de manière sémantiquement correcte. Une erreur fréquente est d'omettre l'attribut `alt` ou de le laisser vide sans raison valable — un `alt` vide (`alt=""`) est réservé aux images purement décoratives. Une autre erreur courante est d'utiliser de simples `` pour regrouper image et légende, ce qui prive le document de sens structurel. Le mental model à retenir : chaque image informative doit avoir un `alt` descriptif et pertinent ; chaque image accompagnée d'une légende doit être encapsulée dans `` + ``. Mini-exemple : `Mon chat Milo au repos.`.
