# Atelier : grille de base à 2 colonnes et 3 lignes

**Rappel théorique** CSS Grid est un système de mise en page bidimensionnel : contrairement à Flexbox qui gère une seule dimension à la fois (ligne OU colonne), Grid permet de contrôler simultanément les lignes ET les colonnes d'une interface. Le conteneur Grid est activé avec `display: grid`. On définit ensuite la structure via `grid-template-columns` pour les colonnes et `grid-template-rows` pour les lignes. Par exemple, `grid-template-columns: 1fr 1fr` crée deux colonnes de largeur égale, où `1fr` représente une fraction de l'espace disponible. La propriété `gap` (ou `column-gap` / `row-gap`) gère l'espacement entre les cellules. Les erreurs fréquentes des débutants : confondre le nombre de tracks (pistes) et le nombre de lignes de grille (grid lines) — une grille à 2 colonnes possède 3 lignes verticales numérotées de 1 à 3 ; oublier de définir les deux axes (colonnes ET lignes) quand on veut un contrôle total ; utiliser des pixels fixes là où `fr` ou `auto` apporteraient plus de flexibilité. Exemple minimal : `.container { display: grid; grid-template-columns: 1fr 1fr; grid-template-rows: auto auto auto; gap: 16px; }` — ce code crée une grille de 2 colonnes et 3 lignes, dans laquelle les 6 enfants directs se placent automatiquement dans l'ordre du flux. --- **Énoncé de l'atelier** Dans cet atelier, vous allez construire une page web simple intégrant une grille CSS de 2 colonnes et 3 lignes, ce qui correspond à exactement 6 cellules.
