# Atelier : créer une barre de navigation flexible

**Rappel théorique** Flexbox (Flexible Box Layout) est un modèle de mise en page CSS conçu pour distribuer l'espace entre les éléments d'un conteneur et gérer leur alignement, même lorsque leurs dimensions sont inconnues ou dynamiques. Le conteneur flex (display: flex) définit le contexte ; ses enfants directs deviennent automatiquement des éléments flex. Deux axes structurent le modèle : l'axe principal (main axis, défini par flex-direction) et l'axe transversal (cross axis, perpendiculaire). Les propriétés clés du conteneur sont justify-content (alignement sur l'axe principal), align-items (alignement sur l'axe transversal), flex-wrap (passage à la ligne) et gap (espacement entre éléments). Côté éléments, flex-grow, flex-shrink et flex-basis (ou la propriété raccourcie flex) contrôlent la façon dont chaque item absorbe ou cède de l'espace. Erreurs fréquentes : appliquer des propriétés flex sur le mauvais élément (confondre conteneur et item), oublier de gérer flex-wrap ce qui provoque un débordement sur petits écrans, et négliger les media queries pour adapter la navigation en mode mobile. Mini-exemple : pour une barre de navigation horizontale basique : nav { display: flex; justify-content: space-between; align-items: center; gap: 1rem; } Ce seul bloc suffit à aligner logo et liens sur une seule ligne, les espaces étant répartis automatiquement. --- **Énoncé de l'atelier** Dans cet atelier, vous allez concevoir et intégrer une barre de navigation complète, responsive et entièrement construite avec Flexbox.
