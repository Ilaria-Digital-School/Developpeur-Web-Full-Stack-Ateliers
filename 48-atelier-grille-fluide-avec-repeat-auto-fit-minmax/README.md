# Atelier : Grille fluide avec repeat(auto-fit, minmax)

**Rappel théorique** CSS Grid offre une fonction puissante pour créer des grilles fluides et responsive sans media queries : la combinaison repeat(auto-fit, minmax()). Décortiquons ce que cela signifie concrètement. - repeat() : répète un gabarit de colonne N fois. Plutôt que d'écrire grid-template-columns: 1fr 1fr 1fr, on écrit repeat(3, 1fr). - auto-fit : au lieu d'un nombre fixe de répétitions, le navigateur calcule automatiquement combien de colonnes peuvent tenir dans le conteneur. Les colonnes vides sont réduites à 0, ce qui permet aux colonnes existantes de s'étirer. - minmax(min, max) : définit une plage de taille pour chaque colonne. minmax(200px, 1fr) signifie « au moins 200px, au plus 1fr de l'espace disponible ». Le mental model : imaginez des tuiles qui se repositionnent toutes seules. Tant qu'il y a de la place pour une tuile de 200px minimum, une nouvelle colonne se crée. Sinon, les tuiles passent à la ligne et s'étirent pour remplir l'espace. Erreurs fréquentes : utiliser auto-fill à la place de auto-fit (les colonnes vides ne disparaissent pas), oublier d'attribuer une valeur fractionnaire comme max dans minmax (résultat : les cartes ne s'étirent pas), ou négliger de définir un gap pour l'espacement entre les cellules. Mini-exemple : .grille { display: grid; grid-template-columns: repeat(auto-fit, minmax(220px, 1fr)); gap: 1.5rem; } Ce code produit une grille dont le nombre de colonnes s'adapte automatiquement à la largeur du conteneur.
