# Mini-projet intermédiaire : page d'accueil portfolio responsive

## Présentation du projet Vous allez concevoir et développer une page d'accueil de portfolio personnel, entièrement responsive, en mobilisant l'ensemble des compétences CSS3, Flexbox et Grid acquises jusqu'ici. Ce projet est un livrable concret, directement valorisable dans votre portfolio de développeur web. ## Contexte En tant que développeur web full stack en devenir, vous devez être capable de concevoir des interfaces front-end propres, accessibles et adaptées à tous les écrans. Cette page d'accueil de portfolio servira de vitrine professionnelle et devra démontrer votre maîtrise du design responsive. ## Structure de la page attendue La page doit comporter les sections suivantes : 1. **Header / Navigation** : barre de navigation fixe ou sticky avec logo/nom et liens d'ancrage (À propos, Projets, Contact). Utiliser Flexbox pour l'alignement. 2. **Section Hero** : présentation visuelle forte (nom, titre, courte phrase d'accroche, bouton CTA). Mise en page centrée avec Flexbox. 3. **Section À propos** : texte de présentation + photo/avatar disposés en deux colonnes avec Grid ou Flexbox. 4. **Section Projets** : grille de cartes projets (au moins 4 cartes) utilisant CSS Grid avec `repeat(auto-fit, minmax(...))` pour un comportement fluide. 5. **Section Compétences** : liste visuelle de compétences techniques, organisée en grille. 6. **Footer** : liens sociaux, copyright. Mise en page avec Flexbox. ## Contraintes techniques obligatoires - Utiliser **CSS Grid** pour la mise en page globale et la section Projets. - Utiliser **Flexbox** pour la navigation, le Hero et le Footer. - Implémenter au minimum **2 Media Queries** : une rupture mobile (≤ 600px)...
