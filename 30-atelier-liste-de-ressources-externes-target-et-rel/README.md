# Atelier : liste de ressources externes (target et rel)

**Rappel théorique** La balise `` est l'élément fondamental pour créer des liens en HTML. Vous avez déjà pratiqué les ancres internes lors de l'atelier précédent. Ici, on s'attaque aux liens externes, c'est-à-dire des liens pointant vers des sites tiers (hors de votre domaine). Deux attributs sont essentiels dans ce contexte : - **`target`** : définit où s'ouvre le lien. La valeur `_blank` ouvre dans un nouvel onglet ou une nouvelle fenêtre. C'est la pratique standard pour les liens externes, afin de ne pas faire quitter votre page à l'utilisateur. - **`rel`** : définit la relation entre votre page et la page cible. Pour les liens externes ouverts avec `target="_blank"`, la valeur **obligatoire** est `rel="noopener noreferrer"`. Pourquoi ? Sans `noopener`, la page cible peut accéder à votre fenêtre via `window.opener`, ce qui constitue une faille de sécurité. `noreferrer` empêche en plus d'envoyer des informations de provenance (référent HTTP) à la page cible. **Erreur fréquente** : utiliser `target="_blank"` sans `rel="noopener noreferrer"`. Les navigateurs modernes ont des protections, mais la bonne pratique reste d'écrire les deux attributs systématiquement. **Mini-exemple** : ```html MDN Web Docs ``` Enfin, les liens externes s'intègrent parfaitement dans des listes HTML (``, ``), ce que vous avez maîtrisé lors des ateliers précédents.
